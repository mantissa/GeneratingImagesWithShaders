# GeneratingImagesWithShaders
A GLSL Shader Programming Workshop at [InterAccess](https://interaccess.org/)\
November 17 & 19 2020

# Outline
* Shader components: inputs, outputs, main() function
* RGB color & HSB color 
* Common functions, custom functions
* 2D shapes
* Animation
* Randomness & Noise
* Grids & Patterns
